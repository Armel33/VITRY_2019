Sélectionner en jQuery

Le coeur de jQuery se trouve dans la sélection d'éléments pour ensuite exécuter des méthodes dessus.

$(selecteur) accepte en argument tout sélecteur CSS, jQuery nous renvoie ensuite une collection d'un ou plusieurs éléments DOM enveloppés dans un objet jQuery.

$('div') séléctionne tous les divs $('.titre') séléctionne tous les éléments qui ont la class="titre" $('#boite') sélectionne l'unique élément qui a l'id="boite"

Cette enveloppe jQuery est celle qui donne accès à toutes les fonctionnalités de jQuery

  • Rajouter des classes
  • Modifier le contenu HTML
  • Afficher ou cacher des éléments
  • Les animer etc.

Tous les éléments sélectionnés en javascript pur (document.getElementById, querySelector etc) sont des éléments DOM et tous ceux sélectionnés avec $('...') sont des éléments DOM dans une enveloppe jQuery

jQuery dispose de plusieurs méthodes pour parcourir l'arbre du document (le DOM)

  • children() parcourt les enfants
  • find() parcourt les descendants
  • next() renvoie le voisin suivant
  • prev() renvoie le voisin précédent
  • parent() renvoie le parent
  • eq() renvoie, parmi une liste d'éléments, celui qui se trove à l'index fourni

Les getters et setters, ajouter, retirer des classes et savoir si on a déjà des class

  • addClass() : ajoute une class
  • removeClass() : retire une class
  • toogleClass() : interrupteur ajoute / retire une class
  • hasClass() : tester l'existance d'une class
  • is() : tester si on a plusieurs classes

$('#pierre').removeClass('vert grand').addClass('rouge petit');

On peut dire à une fonction de s'éxécuter quand l'utilisateur clique sur un bouton ou bien quand il scrolle la fenêtre ou bien encore quand il appuie sur une touche du clavier

    mouseleave : lorsque la souris sort de cette zone

    keydown : une de clavier a été enfoncée
    keyup : une touche de clavier a été relachée
  • window (la fenêtre du navigateur)
    scroll : défilement de la fenêtre
    resize : redimensionnement de la fenêtre
