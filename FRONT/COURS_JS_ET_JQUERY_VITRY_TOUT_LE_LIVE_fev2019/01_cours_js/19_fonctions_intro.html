Les instructions

Un script js est un fichier contenant des instructions, placées les unes à la suite des autres. Chacune de ces instructions peut être vue comme un ordre donné au moteur javascript, une action que vous souhaitez réaliser, à la manière d'une recette de cuisine

  • Faire fondre le beurre
  • Mélanger le beurre et la farine
  • Verser le chocolat petit à petit etc.

Les instructions se suivent et sont effectuées les unes à la suite des autres, Du haut vers le bas (comme en CSS), le browser parse (analyse)dans l'ordre du flux

Chaque instruction DOIT se termioner par un point virgule ; afin d'indiquer au moteur Javascript que cette instruction est terminée

Les fonctions

Les fonctions permettent de regrouper une série d'instructions pour pouvoir les réutiliser. Quand on a besoin du code d'une série d'instructions, plutôt que de réécritre à chaque fois toutes les instructions, on appellera une fonction par son nom (faireUnGateau). Cette fonction contient l'ensemble des instructions à éxécuter.

Par exemple la fonction alert est une fonction native (intégrée au navigateur) qui contient en gros les insgtructions suivantes :

  • Créer le cadre de la boite de dialogue
  • Positionner le cadre dans la fenêtre
  • Crer un background semi transparent sur toute la page
  • Placer un bouton avec le texte OK et un événement click
  • Créer une zone de texte qui affichera le texte de l'art à présenter au visiteur

Nous on a juste besoin d'utiliser la fonction alert() en lui donnant un argument exemple alert('Coucou'). La fonction exécutera toutes les instructions à chaque fois.
On peut utiliser cette fonction alrt quand on veut en la déclenchant avec alert(quelque chose). Ce "Quelque chose" est un ARGUMENT attendu par la fonction.

var maVariable = document.querySelector('h2'); alert('Coucou !');

Les fonctions querySelector et alert sont disponibles dans le navigateur, ce sont des méthodes

Si vous voyez quelque chose qui resszemble à)une variable et qui est suivie de parenthèses c'est une fonction.

Les fonctions prennent souvent des arguments - des bouts de données dont elles ont besoin pour faire leur travail. Les arguments sont placés ENTRE les parenthèses, séparées par des virgules s'il y en a plusieurs.

Par exemple la fonction alert() fait apparaître une popup mais vous devez donneer une chaîne comme argument pour indiquer à la fonction ce que vous sopuhaitez écrire dans la fenêtre.
